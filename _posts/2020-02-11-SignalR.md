## SignalR Nedir ?  ##  
SignalR, microsoft tarafından geliştirilmiş, gerçek zamanlı uygulamalar geliştirmeyi kolaylaştıran teknolojidir. İstemci-sunucu arasında, WebSocket Api ile istemci-sunucu arasındaki iletişimi (çift yönlü) sağlar. Eğer WebSocket kullanılamıyorsa, kodunuzda değişiklik yapmadan,  Open Web Interface (OWIN) aracılığıyla da çalıştırabilirsiniz. Çalışma mantığını hazırladığımız senaryo üzerinde daha detaylı olarak anlatıyor olacağız.
